# Dreamhack Dark Mode

dreamhack.io 웹사이트를 위한 다크모드 브라우저 확장 프로그램입니다.

## 🌙 기능

- dreamhack.io의 모든 페이지에 다크모드 적용
- 눈의 피로를 줄이는 편안한 색상 조합
- 동적으로 로드되는 콘텐츠에도 자동 적용
- 코드 블록, 테이블 등 모든 요소 최적화

## 📦 설치 방법

### Chrome / Edge 브라우저

1. 이 폴더의 위치를 기억해두세요
2. Chrome 또는 Edge 브라우저를 실행합니다
3. 주소창에 다음을 입력합니다:
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`
4. 우측 상단의 **개발자 모드**를 켭니다
5. **압축해제된 확장 프로그램을 로드합니다** 버튼을 클릭합니다
6. 이 프로젝트 폴더를 선택합니다
7. 완료! dreamhack.io에 접속하면 자동으로 다크모드가 적용됩니다

## 🎨 색상 테마

- 배경색: `#1e1e1e` (다크 그레이)
- 텍스트: `#e0e0e0` (밝은 그레이)
- 카드/패널: `#252525` (중간 다크)
- 링크: `#6eb5ff` (블루)
- 버튼: `#3d3d3d` (중간 그레이)

## 🔧 사용자 정의

스타일을 변경하고 싶다면 `dark-mode.css` 파일을 수정하세요.

색상 코드를 변경한 후:
1. 브라우저의 확장 프로그램 페이���로 이동
2. 새로고침 버튼(⟳)을 클릭
3. dreamhack.io 페이지를 새로고침

## 📝 파일 구조

```
dreamhack-dark/
├── manifest.json       # 확장 프로그램 설정 파일
├── dark-mode.css       # 다크모드 스타일시트
├── content.js          # 동적 콘텐츠 처리 스크립트
└── README.md          # 이 파일
```

## ⚠️ 참고사항

- 이 확장 프로그램은 dreamhack.io 도메인에만 작동합니다
- 일부 페이지에서는 스타일이 완벽하게 적용되지 않을 수 있습니다
- 문제가 발생하면 페이지를 새로고침해보세요

## 🐛 문제 해결

### 다크모드가 적용되지 않는 경우
1. 확장 프로그램이 활성화되어 있는지 확인
2. dreamhack.io 페이지를 새로고침
3. 브라우저 콘솔(F12)에서 오류 메시지 확인

### 일부 요소만 밝게 보이는 경우
- 페이지가 동적으로 콘텐츠를 로드하는 중일 수 있습니다
- 몇 초 기다려보거나 페이지를 새로고침하세요
